# 🎮 YT Utils: Utilidades para streamear o jugar

Crea un servicio web con las siguientes opciones:
⚠️ Algunas opciones necesitan que el servidor y el cliente web estén en la misma máquina para funcionar

* **🔊: Configuración de sonidos:**
  - Añade efectos de sonido con su propio identificador
* **🔊: Reproducción de sonidos:**
  - Reproduce los sonidos mediante el identificador definido
*📸; Reconocimiento de texto por captura de pantalla
  - Haz una captura de pantalla del texto que quieras reconocer y selecciona la opción "Reconocer texto por captura". Leerá el texto de la imagen y podrás copiarlo al portapapeles.
  - También puedes traducir un texto mediante una captura. Símplemente haz una captura y selecciona la opción "Traducir texto por captura". Esto leerá el texto de la imagen y lo traducirá al idioma seleccionado
* **Más cosas personalizadas!!**
